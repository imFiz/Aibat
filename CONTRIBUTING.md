# Contributing to X-Booster (Aibat)

First off, thank you for considering contributing to X-Booster! It's people like you that make X-Booster such a great tool.

## Where do I go from here?

If you've noticed a bug or have a question, [search the issue tracker](https://github.com/imFiz/Aibat/issues) to see if someone else has already created a ticket. If not, go ahead and [make one](https://github.com/imFiz/Aibat/issues/new)!

## Fork & create a branch

If this is something you think you can fix, then [fork X-Booster](https://github.com/imFiz/Aibat/fork) and create a branch with a descriptive name.

## Implement your fix or feature

At this point, you're ready to make your changes. Feel free to ask for help; everyone is a beginner at first.

## Get the style right

Your patch should follow the same conventions & pass the same linters as the rest of the project.

## Make a Pull Request

At this point, you should switch back to your master branch and make sure it's up to date with X-Booster's master branch.
Once you have pushed your changes to your fork, simply open a Pull Request!
